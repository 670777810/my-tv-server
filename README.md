# 我的电视·服务

启动一个本地的视频源服务。为了更好的稳定性、隐私性、便捷性，可以把一个视频源文件发布到一个本地地址，添加到本地视频播放器。

[my-tv-server](https://github.com/lizongying/my-tv-server)

## 下载

[releases](https://github.com/lizongying/my-tv-0/releases/latest)

## 用法

1. 下载程序
2. 启动程序
3. 进入网址
4. 上传文件
5. 配置地址

```shell
./my-tv-server
```

![image](./screenshots/img.png)

## 测试

```shell
go run ./cmd/server/*
```

make
